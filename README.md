beers.rb
========

From my lightning talk in Railsberry, Krakow.

Usage
-----

    % irb -I.
    > require 'beers'
    > 3.beers
    > 3.beers + 1.beer
    > 5.beers.compact
    > Random.beers
    > Random.beers(5)
    > 6.pack
    > 24.pack
    > Time.now.beer?
